# git branch

1. git branch

列出所有本地分支

2. git branch `<branch-name>`

创建一个新分支

```bash

```

3. git branch -d `<branch-name>`

删除本地分支（安全删除）

- 安全删除 这个分支的修改是否已经合并（merge）到当前分支或其他分支
- 即只能删除一个已合并的分支
- 强行删除（危险操作）: git branch -D `<branch-name>`
- 删除远程分支要用 git push origin --delete `<branch-name>`

4. git checkout `<branch-name>`

切换到指定分支

5. git switch `<branch-name>`

切换到指定分支

- git checkout：万能但复杂（历史遗留）
- git switch：专门切分支（更安全、更清晰）

6. git checkout -b `<branch-name>`

创建并切换到新分支
